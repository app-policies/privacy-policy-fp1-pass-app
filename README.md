# FP1級Pass プライバシーポリシー

施行日: 2026-06-14

FP1級Pass（以下「本アプリ」）は、学習機能、広告表示、アプリ内課金、利用状況の把握のために、必要な範囲で情報を取り扱います。

## 取得する情報

本アプリでは、以下の情報を取り扱う場合があります。

- 学習履歴、正誤履歴、設定情報
- 広告表示に必要な識別子や端末情報
- アプリ内課金の状態
- クラッシュ情報、利用状況に関する情報

## 利用目的

取得した情報は、以下の目的で利用します。

- 学習機能の提供
- 誤答復習、学習履歴表示などの機能提供
- 広告表示
- アプリ内課金の確認
- 不具合調査、品質改善

## 第三者サービス

本アプリでは、以下の第三者サービスを利用する場合があります。

- Google Mobile Ads
- RevenueCat
- Apple

各サービスで取り扱われる情報は、各サービス提供者のプライバシーポリシーに従って管理されます。
