# FP1級Pass プライバシーポリシー

施行日: 2026-06-14

FP1級Pass（以下「本アプリ」）は、学習機能、広告表示、アプリ内課金、利用状況の把握のために、必要な範囲で情報を取り扱います。

## 取得する情報

本アプリでは、以下の情報を取り扱う場合があります。

- 学習履歴、正誤履歴、設定情報
- 広告表示に必要な識別子や端末情報
- アプリ内課金の状態
- 利用状況に関する情報

## 利用目的

取得した情報は、以下の目的で利用します。

- 学習機能の提供
- 誤答復習、学習履歴表示などの機能提供
- 広告表示
- アプリ内課金の確認
- 利用状況の分析、品質改善

## 第三者サービス

本アプリでは、以下の第三者サービスを利用する場合があります。

- Google Mobile Ads
- Firebase Analytics（利用状況の分析のみ）
- RevenueCat
- Apple

各サービスで取り扱われる情報は、各サービス提供者のプライバシーポリシーに従って管理されます。

Firebase Analytics は、画面遷移や学習開始、回答、学習完了、購入導線の利用状況などを把握し、アプリの品質改善に役立てるために使用します。問題文、選択肢、解説本文、氏名、メールアドレス、個別相談内容、購入取引ID、広告ID、Firebase App ID、RevenueCat SDK Key などは送信しません。

本アプリでは、Firebase Authentication、Firestore、Realtime Database、Cloud Functions、Storage、Remote Config、Crashlytics、Cloud Messaging、App Check は使用していません。
